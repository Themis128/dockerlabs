## Important Notes

1. **Directory Listings vs Direct Files**:
   - Raspberry Pi OS URLs point to directory listings. Your download script must parse the HTML to find the actual .img.xz file.
   - Most other OS images are direct file URLs.

2. **Version Numbers**:
   - Many URLs contain version numbers that change over time. Consider implementing version detection or using "latest" symlinks where available.

3. **File Formats**:
   - Most images are compressed: `.img.xz`, `.img.gz`, or `.zip`
   - Some are uncompressed: `.img`
   - Arch Linux ARM uses: `.tar.gz`

4. **Verification**:
   - Always verify checksums (SHA256) when available
   - Official sources should provide checksum files alongside images

5. **Raspberry Pi Model Compatibility**:
   - Some images are specific to Pi 4, Pi 5, or other models
   - Check the OS documentation for model compatibility

## URL Pattern Reference

### Raspberry Pi OS Pattern
```
https://downloads.raspberrypi.org/{variant}/images/{variant}-latest/
```
Where `{variant}` is one of:
- `raspios_lite_armhf` (32-bit Lite)
- `raspios_armhf` (32-bit Desktop)
- `raspios_full_armhf` (32-bit Full)
- `raspios_lite_arm64` (64-bit Lite)
- `raspios_arm64` (64-bit Desktop)
- `raspios_full_arm64` (64-bit Full)

### Ubuntu Pattern
```
https://cdimage.ubuntu.com/releases/{version}/release/ubuntu-{version}-preinstalled-{type}-arm64+raspi.img.xz
```
Where:
- `{version}` is the Ubuntu version (e.g., `24.04`, `22.04`)
- `{type}` is either `server` or `desktop`

### Debian Pattern
```
https://raspi.debian.net/tested-images/current/arm64/images/debian-{version}-arm64-netinst.img.xz
```
Where `{version}` is the Debian version (e.g., `12.7.0`, `11.9.0`)
